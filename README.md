# Snake Game

A recreation of the classic Snake game built with 
Python's Turtle graphics library.

## Technologies Used
- Python 3
- Turtle (built-in Python graphics library)

## How to Run It
1. Make sure you have Python 3 installed
2. Download ALL these files into the same folder:
   - main.py
   - snake.py
   - food.py
   - scoreboard.py
3. Open your terminal in that folder
4. Run: python main.py

## How to Play
- Use **Arrow Keys** to control the snake
- Eat food to grow longer and increase your score
- Avoid hitting the walls or your own tail
- Game resets automatically if you crash

## Features
- Classic snake movement and controls
- Food spawns randomly on the screen
- Score increases each time food is eaten
- High score tracked and reset on collision
- Collision detection with walls and snake's own tail
